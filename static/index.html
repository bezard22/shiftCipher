Definition
x Plaintext
y Ciphertext
α alphabet
k key
len ( α ) length of alphabet
e ( x ) Encryption
d ( y ) Decryption
e ( x ) = y ≡ x + k mod len ( α )
d ( y ) = x ≡ y − k mod len ( α )
Example
Given:
x = hello (7,4,11,11,14)
α = abcdefghijklmnopqrstuvwxyz
k = 3
len ( α ) = 26
Encryption
e ( 7"h" ) ≡ 7"h" + 3 mod 26 = 10"k"
e ( 4"e" ) ≡ 4"e" + 3 mod 26 = 7"h"
e ( 11"l" ) ≡ 11"l" + 3 mod 26 = 14"o"
e ( 11"l" ) ≡ 11"l" + 3 mod 26 = 14"o"
e ( 14"o" ) ≡ 14"o" + 3 mod 26 = 17"r"
y = khoor (10,7,14,14,17)
Decryption
d ( 10"k" ) ≡ 10"k" − 3 mod 26 = 7"h"
d ( 7"h" ) ≡ 7"h" − 3 mod 26 = 4"e"
d ( 14"o" ) ≡ 14"o" − 3 mod 26 = 11"l"
d ( 14"o" ) ≡ 14"o" − 3 mod 26 = 11"l"
d ( 17"r" ) ≡ 17"r" − 3 mod 26 = 14"o"
x = hello (7,4,11,11,14)
